# 微信小程序－侧滑布局

### 说明：

实现侧滑布局功能，特色：
- 侧滑时内容区缩放
- 拖拽侧滑

### 数据接口:

使用本地数据

### 目录结构：

- images — 存放项目图片文件
- page — 存放项目页面文件
- utils — 存放项目日期格式化等文件

### 开发环境：

微信web开发者工具 v0.11.112301
